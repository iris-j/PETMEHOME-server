# PETMEHOME-server
express server for Mobile Internet course project
### 目录结构 
```
.
├── config.js 端口及数据库配置
├── controller 对user表的一些操作
│   └── user_controller.js
├── middleware passport中间件
│   └── passport.js
├── models 数据库表
│   ├── lost.js
│   ├── message.js
│   ├── pet.js
│   └── user.js
├── package-lock.json
├── package.json
├── routes.js api路由
└── server.js 主文件
```
